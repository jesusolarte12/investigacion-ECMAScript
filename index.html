Guía Tutorial de ECMAScript

¿Qué es ECMAScript?

ECMAScript es una especificación estándar desarrollada por Ecma International que define cómo debe funcionar el lenguaje JavaScript.

Su objetivo es asegurar que los navegadores implementen JavaScript de forma uniforme.

¿Cómo se utiliza ECMAScript?

  • Los navegadores interpretan JavaScript según la especificación ECMAScript.
  • Se utilizan herramientas modernas como Babel y Webpack para escribir código con la última versión de ECMAScript.
  • Se aplican reglas de estilo con ESLint y Prettier.

Versiones de ECMAScript

Versión Año Novedades principales

ES2018 2018 Spread en objetos
ES2019 2019 flat(), trimStart(), fromEntries()
ES2020 2020 ??, ?. (optional chaining)
ES2021 2021 replaceAll(), Promise.any()
ES2022 2022 at(), clases privadas
ES2023 2023 toSorted(), findLast()

Diferencias Notorias entre Versiones

ES5

"use strict";
let numeros = [1, 2, 3];
numeros.forEach(n => console.log(n));

ES6 (2015)

const saludar = nombre => `Hola, ${nombre}`;
console.log(saludar("Ana"));

ES2017

async function cargarDatos() {
  const respuesta = await fetch("https://api.example.com");
  const datos = await respuesta.json();
  console.log(datos);
}

ES2020

let usuario = null;
console.log(usuario?.nombre ?? "Anónimo");

ES2022

let arr = [1, 2, 3];
console.log(arr.at(-1)); // 3

class Persona {
  #secreto = "123";
  mostrarSecreto() {
    return this.#secreto;
  }
}

Conclusión

ECMAScript evoluciona constantemente para ofrecer nuevas funcionalidades y mejorar la experiencia de desarrollo en JavaScript.

Como desarrollador, es fundamental estar al tanto de estas versiones para escribir código moderno, eficiente y compatible.
